Hexo 安装部署及主题更改笔记

一、Hexo 简介

Hexo 是一款基于 Node.js 的快速、简单且强大的静态博客框架。它通过 Markdown 文件生成静态网页，支持多种插件和主题，并且易于部署到各类平台上。
